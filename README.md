# project-mb
**Фамилия**: Волохов
**Группа**: МПБ-301-О-01
# Краткое описание
Этот проект представляет собой простой API, написанный на Flask, который реализует аутентификацию с использованием JSON Web Token (JWT) и базу данных для хранения пользователей. API позволяет создавать пользователей и защищать эндпоинты с помощью JWT.
# Стек

- **Язык**: Python
- **Фреймворк**: Flask
- **Ключевые библиотеки**:
  - Flask-SQLAlchemy
  - Flask-JWT-Extended
  - Flask-Migrate

# Запуск
- **Для запуска системы аутентификации одновременно запустите main.py и api_test.py**
